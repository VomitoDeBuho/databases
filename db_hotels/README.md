# DB_Hotels Sample Database

This document describes the DB_Hotels database

db_hotels és una base de dades creada per [Robert Ventura Vall-llovera](https://www.linkedin.com/in/robertvallllovera/), professor de cicles formatius de grau superior d'informàtica per la Generalitat de Catalunya.
Aquesta base de dades va ser creada per a usos didàctics en assignatures com gestió de base de dades, administració de base de dades.

Aquesta base de dades vol simular una BD de reserves d'hotels a on les taules principals són:

*Paisos
*Poblacions
*Hotels
*Habitacions
*Clients
*Reserves
